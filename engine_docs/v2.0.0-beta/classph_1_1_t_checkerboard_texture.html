Detailed Description

template<typename OutputType>
class ph::TCheckerboardTexture< OutputType >

Texture representing checker patterns.

A texture that is able to represent two inner textures alternately, producing pattern similar to a chessboard. We name the cells in the checkerboard in following manners:

v +--—+--—+--—+ | even| odd| even| +--—+--—+--—+ | odd| even| odd| +--—+--—+--—+ | even| odd| even| +--—+--—+--—+ u
