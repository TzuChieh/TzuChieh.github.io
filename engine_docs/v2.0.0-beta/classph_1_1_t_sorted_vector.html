Detailed Description

template<typename ValueType, typename IsLess = std::less<ValueType>>
class ph::TSortedVector< ValueType, IsLess >

A sorted container that offers fast value lookup & iteration, while modifications are slow. The container has continuous memory allocation. The values are stored in ascending order.
