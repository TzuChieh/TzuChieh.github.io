void ph::SdlParser::enter ( std::string_view  commandSegment,
SceneDescription &  out_scene 
)

Enters a segment of command and parse it once a command is complete.

The command segment must be valid. The method will cache the command segment in subsequent calls until a command is complete, at which point the command will be parsed and the result will act on the provided scene.

A valid command segment is a portion of a complete command that do not break any keyword or symbol of SDL. For example, you can break any opening/closing braces, but you cannot break a type name.

Parameters
commandSegmentA valid segment of command.
[out]out_sceneThe target for parsed results.
