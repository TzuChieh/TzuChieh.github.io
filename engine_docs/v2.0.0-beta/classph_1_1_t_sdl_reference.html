Detailed Description

template<typename T, typename Owner>
class ph::TSdlReference< T, Owner >

A value that points to a SDL resource.

Template Parameters
TType of the referenced SDL resource.
OwnerOwner type of T. Note that Owner can be any type (not necessarily a SDL resource).

Implementations are highly encouraged to throw SdlLoadError if the loading process is not successful. This will allow things such as automatic fallback to work according to field policies.

Convert actual value back to SDL value.

Saving a loaded value as SDL value should rarely fail–as loaded value has been properly handled by the loading process already. In case of failure, throw SdlSaveError and provide detailed reason describing the event.
