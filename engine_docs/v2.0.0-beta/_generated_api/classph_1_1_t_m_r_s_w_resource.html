template<typename Resource>
class TMRSWResource : private ph::INoCopyAndMove

A resource wrapper that facilitates Multiple-Readers Single-Writer (MRSW) usages.

Concurrent access is allowed for read-only operations, while write operations are performed only when exclusive access is acquired.
