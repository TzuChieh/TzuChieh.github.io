Function ph::parallel_work

Function Documentation

inline void ph::parallel_work(const std::size_t totalWorkSize, const std::size_t numWorkers, const std::function<void(std::size_t workerIdx, std::size_t workBegin, std::size_t workEnd)> &work)

Set the parametric distance where the ray ends.

Runs specified works in parallel. The function will block calling thread until all works are complete. Note that 0-sized works are not executed (work is not called in this case).

Parameters

  • totalWorkSize – Total amount of work.

  • numWorkers – Number of workers running the works.

  • work – The actual work that is going to be executed; where the index range [workBegin, workEnd) is specified as input, as well as the index of the executing worker.
