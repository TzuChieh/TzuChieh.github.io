void listAllResources(std::vector<const ISdlResource*> *out_resources, std::vector<std::string_view> *out_resourceNames = nullptr) const

Get a list of all resources in the scene.

Does not involve any change in ownership. Useful for operations that simply needs to iterate over all resources.

Note

Changing the state of the scene may invalidate the returned resource pointers as well as the returned names.

Parameters

  • out_resources – Storage for all resources.

  • out_resourceNames – Storage for all resource names (1-to-1 mapping to out_resources). The memory of resource names are backed by this SceneDescription. Left unspecified (nullptr) if resource names are not required.
