template<typename KeyType, typename ValueType, typename IsLess = std::less<KeyType>>
class TSortedMap

A sorted container that offers fast value lookup & iteration, while modifications are slow. The container has continuous memory allocation. The values are stored in ascending order according to their keys.

The value associated with the key. If there are more than 1 values mapped to the key, the pointer to the first one will be returned. Empty if no mapped value found.
