Detailed Description

template<typename T>
class ph::math::TAABB3D< T >

A 3-D Axis-Aligned Bounding Box.

A construct that have a cuboid-shaped boundary. An AABB, depending on the usage, may represent a point, a layer of skin, or a volume.

Template Parameters
TArithmetic type used for the calculation.

inline

Checks whether a line segment intersects this volume.

Returned boolean value indicates whether the ray is intersecting with the AABB's volume or not. If there's an intersection, the near and far hit distances will be returned via (out_rayNearHitDist, out_rayFarHitDist); if the starting point of the ray is inside the AABB, near hit distance will be the ray's t_min since volume intersection starts immediately on the ray's starting point (the ending point of the ray follows the same logic).

Parameters
[out]out_nearHitTParametric distance where the line segment starts to intersect this volume.
[out]out_farHitTParametric distance where the line segment no longer intersects this volume.
Returns
True if intersection is found, otherwise false. When false is returned, the value of out_nearHitT and out_farHitT is unspecified.
