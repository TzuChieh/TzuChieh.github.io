Detailed Description

template<typename OutputType>
class ph::TConstantTexture< OutputType >

Texture storing only one single constant. This texture provides only a constant value. Input color usages are ignored. For color-related constants, consider using TConstantTristimulusTexture or TConstantSpectralTexture; otherwise, it is recommended to bake color-related side effects (e.g., color usages) into the constant.
