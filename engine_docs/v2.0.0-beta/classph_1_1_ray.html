Detailed Description

Representing a ray in space.

Points \( \overrightarrow{P} \) on a ray can be modeled using the equation

\[ \overrightarrow{P}=\overrightarrow{O}+t\overrightarrow{D} \]

where \( \overrightarrow{O} \) is ray origin and \( \overrightarrow{D} \) is ray direction, and \( t \) is a parameter in \( [t_{min}, t_{max}] \). This class also records the time the ray was set on. Note the direction vector of the ray does not need to be normalized.
