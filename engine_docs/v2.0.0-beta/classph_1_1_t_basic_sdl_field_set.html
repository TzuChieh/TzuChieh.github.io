Detailed Description

template<typename BaseFieldType, std::size_t MAX_FIELDS = PH_SDL_MAX_FIELDS>
class ph::TBasicSdlFieldSet< BaseFieldType, MAX_FIELDS >

A set of fields, with basic functionalities.

This class accepts polymorphic field types. This class finds a field using brute-force method.
