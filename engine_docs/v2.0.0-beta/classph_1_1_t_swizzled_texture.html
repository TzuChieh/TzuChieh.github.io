Detailed Description

template<typename InputT, typename OutputT, std::size_t OUTPUT_N>
class ph::TSwizzledTexture< InputT, OutputT, OUTPUT_N >

Swizzle an array-like input type to the other array-like output type. The output object can have size greater or lesser than the input object (talking about max index).

Template Parameters
OUTPUT_NSize of the array-like object.
