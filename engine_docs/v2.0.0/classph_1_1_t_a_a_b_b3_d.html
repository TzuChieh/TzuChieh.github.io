template<typename T>
ph::TAABB3D class

A 3-D Axis-Aligned Bounding Box.

Template parameters
T Arithmetic type used for the calculation.

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

A construct that have a cuboid-shaped boundary. An AABB, depending on the usage, may represent a point, a layer of skin, or a volume.
